# CurationsLA — Public Roadmap

Welcome to CurationsLA's public roadmap. The purpose is to be transparent about what we're building, collect community feedback, and surface priorities.

## How to use this roadmap
- Read the "Now / In-Progress" and "Next / Planned" sections to see what we're working on.
- Upvote using the Google Form (link below) — your vote helps us prioritize.
- Submit new ideas via the Google Form or open a GitHub issue using the provided template.

## How we prioritize
We consider:
1. Community votes
2. Community impact and reach
3. Implementation effort and feasibility
4. Strategic alignment with CurationsLA goals

## Suggested repository workflow (manual)
1. Community submits ideas via Google Form.
2. Maintainers review new submissions weekly and create GitHub Issues for accepted items using the roadmap issue template.
3. Link the issue number into ROADMAP.md and assign a milestone/label.
4. Use Google Sheet tallies and issue comments to inform prioritization decisions.
