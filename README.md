# curationsdev/roadmap

Public roadmap and feature requests for CurationsLA.

## Purpose
- Provide a single, public view of CurationsLA's product roadmap
- Accept community submissions and upvotes from people who may not use GitHub
- Keep communications public, simple, and transparent

## How to publish GitHub Pages
1. In the repository Settings → Pages:
   - Under "Build and deployment", set Source to the `main` branch and folder `/docs`
   - Save. GitHub Pages will publish the site (it may take a minute to build)
2. The site will be available at a URL shown in Pages settings (e.g. `https://curationsdev.github.io/roadmap`)
